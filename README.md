# Sales Data Analysis using Python (Pandas)

This project analyzes a simple sales dataset using Python and the Pandas library.  
It demonstrates how to load data, clean it, analyze sales metrics, and generate a basic report.

---

## Project Structure

sales-data-analysis  
├── sales_analysis.py  
├── sales_data.csv  
└── README.md  

---

## Technologies Used

- Python 3  
- Pandas  
- Python IDLE  

---

## Project Description

The goal of this project is to perform basic data analysis on sales data.  
The program loads a CSV file, explores the dataset, handles missing values, removes duplicates, and calculates important sales metrics.

---

## Features

- Load sales data from CSV file  
- Display first few rows of the dataset  
- Check number of rows, columns, and data types  
- Handle missing values  
- Remove duplicate records  
- Calculate:
  - Total revenue  
  - Average sales  
  - Highest and lowest sales  
  - Best-selling product  
  - Sales by region  

---

## How to Run the Project

1. Install pandas (only once):
   ```
   pip install pandas
   ```

2. Keep both files in the same folder:
   - sales_analysis.py  
   - sales_data.csv  

3. Open Python IDLE.

4. Open `sales_analysis.py` and press **F5** to run the program.

---

## Output

The program prints:
- Dataset overview  
- Missing value information  
- Cleaned data confirmation  
- Final sales analysis report
